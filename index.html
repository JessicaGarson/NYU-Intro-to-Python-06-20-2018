Expressions

Expressions consist of values (such as 2) and operators (such as +), and they can always evaluate (that is, reduce) down to a single value.

Operators

  • ** Exponent
  • % Modulus/remainder
  • // Integer division/floored quotient
  • / Division
  • * Multiplication
  • - Subtraction
  • + Addition

Data Types in Python

  1. Integers such 1, 23, -32, 24, 234
  2. Floating point numbers or floats such as 1.23, 23.23, -4.32, 22.6
  3. Strings such as 'rwewe', '2si', 'hello', 'anything really'

Variables

Variables are containers for information. You can store any of the three data types in variables.

We've been printing, whats that about?

The print() function will display the string value inside the parentheses on the screen.

So what exactly is string?

Strings are combinations of characters. They can be letters, numbers, punctuation.

What happens if my string is very long?

You can use triple quotes for long strings.

Special characters

  • \n new line
  • \t tab

Slicing

Slicing allows us to grab sections of the variable we are working with.

String formatting

We can add variables into strings to make them more reusable and robust.

Built functions for strings

Strings have some built in functions that do interesting things. We'll demo a few.

Conditionals

We use conditionals to compare values.

Operators for Comparsion

  • == These two things are equal
  • != NOT! equal to
  • > Greater than
  • < Less than
  • >= Greater than or equal to
  • <= Less than or equal to

If Statement

The if statement evaluates whether a statement is true or false, and run code only in the case that the statement is true.

Else

If none of the conditions meet. Than do this.

Elif

The elif statement allows you to check multiple expressions to see if one is true. There can be multiple elif statements in a program.
